World Modifiers Cheatsheet

A quick-reference for all Valheim world modifier commands. Part of the Nort-Sun Gaming Community.

🔍

⚙️ How to Use World Modifiers

1
Open console — press F5 in-game (enable cheats first)
2
Enable cheats — type devcommands and press Enter
3
Type the command — copy any command below and paste it in the console
4
Reload the world — some modifiers require a world reload to take effect
⚔️ Combat
Player Damage
Adjust how much damage players deal to enemies.
setkey PlayerDamage <value>
veryhard normal easy veryeasy
Enemy Damage
Change how much damage enemies deal to players.
setkey EnemyDamage <value>
veryhard normal easy veryeasy
Death Penalty
Control what you lose on death — skills, items, or nothing.
setkey DeathPenalty <value>
hardcore normal easy casual
PvP
Toggle player-vs-player damage on or off globally.
setkey PVP <value>
true false
👹 Enemies & Raids
Enemy Level
Set the level scaling of spawned enemies.
setkey EnemyLevel <value>
veryhard normal easy
No Raids
Disable enemy raid events entirely on this world.
setkey noraids
No more raids!
Passive Enemies
Make all enemies passive — they will not attack unless attacked first.
setkey passivemobs
Peaceful mode
No Mob Spawn Zone
Prevents enemies from spawning within a set radius of your spawn point.
setkey nospawn
Safe base!
🪵 Resources & Crafting
Resource Rate
Multiply how many resources you gather from nodes and drops.
setkey ResourceRate <value>
muchless normal more muchmore
Crafting Stations
Toggle whether crafting near a workbench or station is required.
setkey NoCraftingStationRequired
Craft anywhere
Food Consumption
Change how quickly food buffs drain over time.
setkey FoodDrain <value>
veryfast normal slow veryslow
🗺️ World & Exploration
No Portal Item Restriction
Allow all items — including ores and metals — through portals.
setkey noportals
Removes portals entirely
Allow Portal Items
Carry any item through portals, including ores, metals, and eggs.
setkey AllowPortalBypass
All items allowed
Shared Map
Share explored map progress between all players on the server.
setkey SharedMap
Multiplayer friendly
Dungeon Instances
Give each player their own private instance of dungeons instead of sharing.
setkey PlayerInstances
Solo instances
📈 Skills & Progression
Skill Gain Rate
Adjust how fast skills level up during gameplay.
setkey SkillGain <value>
muchless normal more muchmore
Skill Penalty on Death
Reduce or remove skill point loss when you die.
setkey SkillPenaltyOnDeath <value>
normal half none
🛠️ Utility Commands
List Active Keys
Show all currently active world modifier keys in the console.
listkeys
Debug tool
Remove Key
Remove a specific world modifier key to revert it to default.
removekey <keyname>
Resets modifier
Reset All Keys
Remove all custom world modifier keys and reset to default world settings.
resetkeys
Resets everything!
Set Time of Day
Set the time of day. 0 = midnight, 0.5 = midday.
tod <0–1>
tod 0.5 = noon
